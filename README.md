# 在线简历生成器

一个现代化的在线简历生成器，帮助用户快速创建专业的简历。

## 项目概述

在线简历生成器是一个基于Web的应用程序，允许用户通过简单的界面创建、编辑和下载专业简历。用户可以选择不同的模板，自定义样式，并以PDF格式导出简历。

## 功能特性

- 多种简历模板选择
- 实时预览功能
- PDF导出功能
- 响应式设计，支持移动设备
- 简单易用的编辑界面
- 本地存储功能（可选）

## 🛠 技术栈

- React.js - 前端框架
- Node.js - 运行时环境
- HTML5/CSS3 - 网页结构与样式
- JavaScript (ES6+) - 客户端逻辑
- Webpack - 模块打包工具
- Babel - JavaScript编译器
- canvg - SVG到Canvas转换库
- 以及其他依赖库

## 🚀 项目启动步骤

### 前提条件

确保您的系统已安装：
- [Node.js](https://nodejs.org/) (v14.0.0 或更高版本)
- npm 或 yarn 包管理器

### 安装与运行
```text
git clone"xxxxxxxxxx"此处填写clone链接
```
用vscode打开文件夹
新建终端依次输入指令:
```text
cd resume-vue
```
```text
npm run dev
```
ctrl+点击显示链接，即可进入网页

### 使用说明
进入欢迎页面点击页面中间的蓝色按钮“制作一份简历”
进入功能页面后可以根据需要进行简历的完成。
