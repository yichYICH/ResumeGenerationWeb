# 🚀 在线简历生成器 (Online Resume Builder)

> 一个现代化、隐私安全、无需后端的在线简历制作工具。所见即所得，一键导出 PDF。

![项目封面图](path/to/your/screenshot.png) ## 📖 项目介绍

这是一个基于纯前端技术栈（HTML5 + CSS3 + ES6）开发的在线简历生成应用。它旨在帮助用户通过简单的交互界面，快速生成美观、专业的简历。

**核心亮点：**
* **隐私优先**：所有数据仅存储在本地浏览器（LocalStorage），不上传任何服务器，确保用户隐私安全。
* **所见即所得**：左侧编辑，右侧实时预览，支持缩放查看。
* **高度可定制**：支持更换 6 种不同风格的模板、自定义主题色、字体和字号。

## ✨ 主要功能 (Features)

### 1. 强大的编辑器
* **实时预览**：修改表单内容，右侧简历实时更新。
* **模块化管理**：支持动态添加/删除教育经历、工作经验、科研项目、技能点等模块。
* **富文本处理**：自动处理换行格式，支持长文本输入。
* **头像上传**：支持本地图片预览与裁剪（自动适配）。

### 2. 多样化模板系统
内置 6 套精心设计的 CSS 模板，一键切换：
* **现代风 (Modern)**：带图标的时间轴设计，适合互联网行业。
* **简约风 (Classic)**：经典衬线字体，优雅稳重。
* **专业风 (Professional)**：双栏布局，深色侧边栏，适合商务/管理岗。
* **创意风 (Creative)**：独特的视觉元素，适合设计类岗位。
* **极简风 (Minimal)**：极致留白，信息降噪。
* **学术风 (Academic)**：严谨的布局，适合科研/学术申请。

### 3. 个性化设置
* **全局配色**：支持自定义主色调，文字颜色。
* **字体库**：内置多种中英文字体（如思源黑体、Roboto、Times New Roman 等）。
* **布局调整**：支持字体大小无级调节。

### 4. 数据与导出
* **PDF 导出**：集成 `html2pdf.js`，支持 A4 纸张格式高清导出。
* **数据持久化**：自动保存编辑进度到本地。
* **JSON 备份**：支持将简历数据导出为 JSON 文件，或从 JSON 导入恢复数据。
* **演示模式**：提供“一键填充演示数据”功能，方便快速查看效果。

## 🛠️ 技术栈 (Tech Stack)

* **核心语言**: HTML5, CSS3 (CSS Variables, Flexbox, Grid), JavaScript (ES6+)
* **样式处理**: 原生 CSS，未使用 Bootstrap 等重型框架，保证轻量化。
* **图标库**: [RemixIcon](https://remixicon.com/)
* **字体服务**: Google Fonts
* **PDF 生成**: [html2pdf.js](https://github.com/eKoopmans/html2pdf.js)
