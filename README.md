# 在线简历生成器

> 一个现代化、隐私安全、无需后端的在线简历制作工具。所见即所得，一键导出 PDF。

## 项目介绍

这是一个基于 vue+css 开发的在线简历生成应用。它旨在帮助用户通过简单的交互界面，快速生成美观、专业的简历。

## 项目启动步骤

### 前提条件

确保您的系统已安装：
- [Node.js](https://nodejs.org/) (v14.0.0 或更高版本)
- npm 或 yarn 包管理器

### 安装与运行
```text
git clone"xxxxxxxxxx"此处填写clone链接
```
用vscode打开文件夹
新建终端依次输入指令:
```text
cd resume-vue
```
```text
npm run dev
```
ctrl+点击显示链接，即可进入网页

如果需要使用ai润色功能  
请在\resume-vue\src\utils\aiService.js代码文件，按注释要求更换自己的api。

### 使用说明
进入欢迎页面点击页面中间的蓝色按钮“制作一份简历”  
进入功能页面后可以根据需要进行简历的完成。  
再输入内容后可以按输入框右上角的紫色按钮进行ai润色语句。
